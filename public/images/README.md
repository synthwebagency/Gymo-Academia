# Gymo Academia Images

This folder contains the images used in the Gymo Academia application.

## How to add images on GitHub:
1.  Upload your image files directly into this `public/images/` folder in your GitHub repository.
2.  Ensure the filenames match the ones used in the code (e.g., `logo.png`, `boxe.jpg`).
3.  The application will automatically find them at `/images/your-image.jpg` after building.
